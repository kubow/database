    • 1. Dynamic histograms

    Dynamic histograms allow you to specify columns for which a new range cell is dynamically maintained at the end of an existing histogram.

    For DML operations on this column, the new range cell includes the maximum value and dynamically maintains a row count.

    The boundaries of the original cells of the histogram remain unchanged, but the weights of the original cells are proportionally decreased as the weight of the new cell is increased.

    update statistics  (, ...)
... 
     [using dynamic_histogram [ on | off ] 
...

    2. Mount and unmount encrypted databases

    mount and unmount encrypted databases the same as mounting unencrypted databases

    The master key and database encryption keys stored in the master database must be the same.

    3. Using select into in multistatement transactions

    SAP ASE allows you to use the select into command against permanent and temporary tables in multistatement transactions

    4. Create or alter databases to use compressed indexes

    Use the create database ... index_compression parameter to create databases that include compressed indexes.

    Use the alter database ... set index_compression to enable or disable index compression on existing databases.
